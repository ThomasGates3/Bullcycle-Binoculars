🚀 DEPLOY IN 5 STEPS (Takes ~30 minutes):

1️⃣  CREATE AWS ACCOUNT
    Go to: https://aws.amazon.com
    Time: 5 minutes

2️⃣  CREATE IAM USER
    AWS Console → IAM → Users → Create User
    Name: bullcycle-github-deploy
    Attach: AmazonS3FullAccess, CloudFrontFullAccess
    Create access key → Copy both keys
    Time: 3 minutes

3️⃣  ADD GITHUB SECRETS
    GitHub → Repo Settings → Secrets → New Secret
    Add: AWS_ACCESS_KEY_ID = (from step 2)
    Add: AWS_SECRET_ACCESS_KEY = (from step 2)
    Time: 2 minutes

4️⃣  RUN TERRAFORM
    cd /Users/tg3/dev/bullcycle-binoculars/terraform
    terraform init
    terraform plan
    terraform apply (type 'yes')
    Time: 5 minutes
    ⏳ Wait 5-10 minutes for CloudFront to deploy!

5️⃣  TEST IT WORKS
    git add .
    git commit -m "Test deployment"
    git push origin main
    → Watch GitHub Actions → See live URL
    Time: 5 minutes

❓ NEED HELP?

Terraform error?
  → Check terraform/outputs.tf for variable names
  → Ensure AWS credentials are configured

Deployment not working?
  → Check GitHub Actions tab → See error logs
  → Verify AWS_ACCESS_KEY_ID and AWS_SECRET_ACCESS_KEY are in GitHub Secrets

CloudFront shows "Access Denied"?
  → It's normal during first deployment
  → Wait 5-10 minutes for CloudFront to fully deploy
